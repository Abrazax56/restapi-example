## penggunaan
### instalasi 
###### 1. install bun js
• curl
```bash
curl -fsSL https://bun.sh/install | bash
```
• npm
```bash
npm i -g bun
```
###### 2. install dependensi
```bash
bun install
```
###### 3. inisialisasi kompiler
```bash
npx tsc --init
```
###### 4. kompilasi kode Typescript
```bash
npx tsc
npx tsc --watch
```
###### 5. jalankan kode
```bash
bun run ./src/index.ts
```
