## penggunaan
# instalasi 
1. install bun js
• curl
```
curl -fsSL https://bun.sh/install | bash
```
• npm
```
npm i -g bun
```
2. install dependensi
```
bun install
```
3. inisialisasi kompiler
```
npx tsc --init
```
4. kompilasi kode Typescript
```
npx tsc
//jika ingin melihat kompilasi langsung
npx tsc --watch
```
5. jalankan kode
```
bun run ./src/index.ts
```
